# LANES Frontend — Responsive PWA Overhaul

> **NOTE:** We are not changing the overall design of the frontend. We are editing it to make it responsive and PWA-capable. Emojis on the frontend should be replaced with `lucide-react` icons.

## Objective

Transform the existing Next.js frontend into a fully responsive, mobile-first Progressive Web Application (PWA) that delivers a seamless flood-adaptive navigation experience across all device sizes — from smartphones to desktop workstations.
